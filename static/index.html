Scientific Calculator
Supported: +, -, *, /, ^, sqrt(), sin(), cos(), tan(), log() (natural), ln(), log10(), exp(), abs(), factorial(), pi, e
Basic formulas:
Add: $a + b$  |  Subtract: $a - b$  |  Multiply: $a \times b$  |  Divide: $\dfrac{a}{b}$
Power: $a^{b}$  |  Square root: $\sqrt{x}$  |  Log: $\ln(x)$ or $\log_{\text{base}}(x)$
2D Shapes



Formulas:
Rectangle: Area = $w \times h$; Perimeter = $2(w + h)$
Circle: Area = $\pi r^{2}$; Circumference = $2\pi r$
Square: Area = $s^{2}$; Perimeter = $4s$
Triangle: Area = $\tfrac{1}{2} \, b\, h$; Perimeter = sum of sides
3D Shapes







3D Formulas:
Cube: Volume = $s^{3}$; Surface Area = $6s^{2}$
Sphere: Volume = $\tfrac{4}{3}\pi r^{3}$; Surface Area = $4\pi r^{2}$
Cylinder: Volume = $\pi r^{2} h$; Surface Area = $2\pi r(r + h)$
Cuboid: Volume = $l \times w \times h$; Surface Area = $2(lw + lh + wh)$
Prism: Volume = $\text{baseArea} \times \text{height}$; Lateral area = $\text{perimeter} \times \text{height}$
Quadratic Function
Quadratic formula:
Discriminant: $\Delta = b^{2} - 4ac$
Roots: $x = \dfrac{-b \pm \sqrt{\Delta}}{2a}$
(If $\Delta < 0$ there are no real roots; if $\Delta = 0$ one real root)
BMI Calculator
BMI formula:
$$\displaystyle \text{BMI} = \frac{\text{weight (kg)}}{(\text{height (m)})^{2}}$$
$$\displaystyle \text{If height in cm: BMI} = \frac{\text{weight}}{(\text{height (cm)} / 100)^{2}}$$
Statistics

Ungrouped Data

Ungrouped formulas:
Mean: $\bar{x} = \dfrac{\sum x_i}{n}$
Median: middle value of sorted data (or average of two middle values)
Mode: most frequent value
Std Dev (sample): $s = \sqrt{\dfrac{\sum (x_i - \bar{x})^{2}}{n - 1}}$

Grouped Data

Grouped formulas:
Mean: use class midpoints $m_i = L + (i+0.5)h$; $\bar{x} = \dfrac{\sum m_i f_i}{\sum f_i}$
Median: $L + \dfrac{(N/2 - C)}{f_m} \times h$ where $C$ is cumulative freq before median class
Mode (interpolated): $L + \dfrac{(f_m - f_{1})}{2f_m - f_{1} - f_{2}} \times h$
